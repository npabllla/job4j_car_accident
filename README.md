# job4j_car_accident
## Проект "Автонарушители"
### Веб-приложение для добавления и редактирования автонарушений
## Используемые технологии
* Spring (Core, MVC, Data, Security)
* Jsp, Html
* Hibernate
* PostgreSQL
* Log4j, Sl4J
* Maven
* Apache Tomcat
* Travis CI, CheckStyle
